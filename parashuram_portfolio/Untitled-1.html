Projects

Project 1: [Expense Tracker]

The Expense Tracker is a simple yet functional web application designed to help users manage their expenses efficiently. It allows users to add items with their respective amounts, view the list of expenses, and calculate the total amount dynamically.
Key Features:
Add Items: Users can input item names and their amounts, which get displayed in a table format.
Clear All: Quickly clears all entered data for a fresh start.
Total Calculation: Dynamically calculates and displays the total expense.
Technologies Used:
HTML: For the structure and layout of the application.
CSS: Styled the application with a clean and professional interface, including shadows and borders for better visual appeal.
JavaScript: Handles dynamic functionalities such as adding items, updating the table, and calculating the total expense.
Learning Outcomes:
Gained experience in DOM manipulation with JavaScript. Practiced creating interactive and user-friendly interfaces. Improved understanding of form validation and event handling.
